# OpenAI Codex Adapter

- Provider id: `codex`
- Upstream profile: `codex`
- Project path: `.agents/skills`
- Global path in provider matrix: `~/.codex/skills`
- Optional bridge: `AGENTS.md`
- Official docs: <https://developers.openai.com/codex/skills>, <https://developers.openai.com/codex/guides/agents-md>
- Last verified: 2026-06-10

Codex supports portable Agent Skills with `SKILL.md`, `scripts/`, `references/`, and optional `agents/openai.yaml`. The bridge only points to installed skills and does not duplicate skill content.
